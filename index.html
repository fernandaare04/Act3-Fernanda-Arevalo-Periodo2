Texto y Tipografía en CSS

Identificar los diferentes estilos de texto que se pueden configurar usando hojas CSS



En el CSS podemos modificar la apariencia del texto utilizando diferentes propiedades relacionadas con la tipografía y el formato del texto. Estas propiedades permiten controlar la fuente, tamaño, alineación, espaciado y decoración del contenido dentro de una página web.

Estilo Tipográfico

En esta sección se utilizan propiedades relacionadas con la familia tipográfica. La propiedad font-family permite cambiar el tipo de letra del texto, mientras que font-size define el tamaño.

También se utiliza font-weight para hacer el texto más grueso y text-transform para transformar el texto en mayúsculas. Esto ayuda a destacar títulos o información importante.

Alineación y Espaciado

Las propiedades de texto permiten organizar mejor el contenido. Por ejemplo, text-align controla la alineación del texto ya sea a la izquierda, derecha, centrado o justificado.

También podemos usar letter-spacing para aumentar el espacio entre letras y line-height para mejorar la separación entre líneas, facilitando la lectura del contenido.

Decoración del Texto

CSS también permite agregar efectos visuales al texto. La propiedad text-decoration puede subrayar, tachar o resaltar palabras dentro del contenido.

Además, text-shadow permite aplicar sombras al texto para darle más profundidad visual. Esto se utiliza comúnmente en títulos o elementos destacados dentro del diseño web.
